runjar:enhance-ant

Full name:

net.kozelka.maven:runjar-maven-plugin:0.1-SNAPSHOT:enhance-ant

Description:

Enhances existing ANT buildfile with an executable jar with ant in it

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile+runtime.
  • Binds by default to the lifecycle phase: package.

Optional Parameters

Name Type Since Description
antFile String - (no description)
Default value is: main.ant.xml.
antScriptsDirectory File - (no description)
Default value is: ${project.build.scriptSourceDirectory}.
attach boolean - Whether to attach the runnable artifact.
Default value is: true.
classifier String - Classifier to use when attaching - see attach.
Default value is: run.
runjarDirectory File - The working directory where runnable jar's content is created.
Default value is: ${project.build.directory}/runjar.
runnableJar File - Runnable jar file name (output). Its extension is used as the artifact type when attaching to maven lifecycle.
Default value is: ${project.build.directory}/${project.build.finalName}-run.jar.

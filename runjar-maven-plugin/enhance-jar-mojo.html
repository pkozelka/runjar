runjar:enhance-jar

Full name:

net.kozelka.maven:runjar-maven-plugin:0.1-SNAPSHOT:enhance-jar

Description:

Enhances existing JAR (with a Main-Class attribute) to be an executable jar; adds libraries, boot etc.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile+runtime.
  • Binds by default to the lifecycle phase: package.

Optional Parameters

Name Type Since Description
attach boolean - (no description)
Default value is: true.
classifier String - (no description)
Default value is: run.
jar File - Original jar file name (input)
Default value is: ${project.build.directory}/${project.build.finalName}.jar.
mainClass String - the entry point.
Default value is: ${project.groupId}.${project.artifactId}.Main.
runjarDirectory File - (no description)
Default value is: ${project.build.directory}/runjar.
runnableJar File - Runnable jar file name (output). Its extension is used as type when attaching to maven lifecycle.
Default value is: ${project.build.directory}/${project.build.finalName}-run.jar.
